销售管理业务是对厂内销售合同执行过程进行管控。主要涉及销售基础数据维护、销售指标管理、销售合同管理、销售订单管理、投产指令管理、销售发货管理、销售收入确认、应收台账管理、开票与回款管理、销售预警和销售财务综合查询。

业务流程如下：

# $\bullet$  功能描述

销售人员对销售客户类型进行维护。

# $\bullet$  操作说明

# 2.1. 销售指标管理

# 2.4.投产指令管理

# 2.4.1. 销售订单生成投产指令

# $\bullet$  功能描述

销售人员根据订单情况下达投产指令。

# $\bullet$  操作说明

点击“销售管理”  $\rightarrow$  “投产指令管理”  $\rightarrow$  “销售订单生成投产指令”。弹出投产指令下达页面，选中需投产的订单。

是否生成计划：“是”表示下一步会到主计划进行进行投产；“否”表示不进行投产只用来订单发货。

库存冲抵：利用已有库存冲抵，直接销售发货。选择库存冲抵的产品会被占用，只有该订单对的发货才能使用。

库存改装：利用库存已有产品进行改装\复验后再销售发货。选择改装的产品会被占用，同时会自动生成出库单。

选配：针对BOM中有选配件的进行选配。

![](images/83f74fbe81eea508a2734d334e326f6389da92a828190cc6fa338cd99585523b.jpg)
